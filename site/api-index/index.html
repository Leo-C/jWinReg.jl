RegCreateKey(key::AbstractString) ¶

Recursively create specified key. key can be specified using abbreviated form (e.g. 'HKEY_LOCAL_MACHINE' -> 'HKLM').
Return 0 if key is existent, 1 if successful, -1 otherwise

Delete specified key. key can be specified using abbreviated form (e.g. 'HKEY_LOCAL_MACHINE' -> 'HKLM').
